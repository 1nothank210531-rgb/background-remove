# AI 自動背景移除工具 (AI Background Remover)

這是一個基於 Python 與 `rembg` 開源模型的自動化去背腳本，能夠快速將圖片轉換為透明背景的 PNG 格式。

## 🚀 功能特色

* **高品質去背**：採用 AI 模型自動偵測主體，精準分離背景。
* **邊緣平滑優化**：支援 Alpha Matting 技術，讓毛髮或複雜邊緣的處理更自然。
* **簡單易用**：只需設定輸入路徑即可一鍵執行。
